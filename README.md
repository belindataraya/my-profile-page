### What is HTML?

Think of building a house 🏠:
•	HTML is the structure — the walls, roof, doors, and windows
•	CSS is the decoration — the paint, curtains, and furniture

HTML stands for HyperText Markup Language. It tells the browser what to show on a page — headings, paragraphs, images, buttons, links, etc.

CSS stands for Cascading Style Sheets. It tells the browser how things should look — colours, sizes, fonts, spacing, etc.

Without HTML, there's no content. Without CSS, everything looks plain and ugly. Together, they make every website you've ever visited.

## Section 4: Installation & First Steps

How to create your first HTML file:

1.	Open Notepad (Windows)
2.	Type or paste your HTML code
3.	Click File → Save As
4.	Name your file index.html (make sure it ends in .html not .txt)
5.	Save it to your Desktop
6.	Double-click the file — it will open in your browser automatically
7.	Every time you make changes, save the file and refresh the browser to see updates

## Section 5: Related Working Example

This below is my personal profile webpage I built during this project using HTML for structure and CSS for styling. It includes a heading, about me section, hobbies list and a goal statement.

Below is the code in the Notepad saved as index.html.

<!DOCTYPE html>
<html>

  <head>
    <title>My Profile Page</title>
    <style>

      /* This is CSS — it styles the page */

      body {
        font-family: Arial;
        background-color: #fff0f9;
        margin: 40px;
        color: #333333;
      }

      h1 {
        color: #ff6600;
        text-align: center;
      }

      h2 {
        color: #d439b7;
      }

      .profile-box {
        background-color: white;
        padding: 30px;
        border-radius: 10px;
        max-width: 600px;
        margin: 0 auto;
      }

      ul {
        line-height: 2;
      }

    </style>
  </head>

  <body>
    <div class="profile-box">

      <h1>👋 Hi, I am Belinda!</h1>

      <h2>About Me</h2>
      <p>
        I am a student at Moringa School learning
        how to build websites using HTML and CSS.
        with the power of AI.
        I am excited to grow my skills in software
        development!
      </p>

      <h2>My Hobbies</h2>
      <ul>
        <li>🎵 Listening to music</li>
        <li>📚 Reading books</li>
        <li>💻 Learning to code</li>
        <li>⚽ Watching football</li>
      </ul>

      <h2>My Goal</h2>
      <p>
        My goal is to become a confident frontend
        developer and build beautiful websites
        that help people.
      </p>

    </div>
  </body>

</html>

## Section 7: Common Gotchas & Fixes

These are mistakes I made or things that confused me:

•	❌ Forgetting to close a tag — e.g. writing <h1>Hello without </h1> breaks the layout. Always close your tags!

•	❌ Saving as .txt instead of .html — the file won't open as a webpage. Always check the file extension when saving

•	❌ CSS not working — I forgot to put CSS inside <style> tags. The code must be in the right place

•	✅ Fix: When something looks wrong, I pasted my code into Claude and asked "what is wrong with this code?" — it always explained the problem clearly
